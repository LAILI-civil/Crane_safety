Introduction
This project is a computer vision-based risk identification system for tower crane construction. It aims to automatically analyze images or videos from construction sites to detect potential safety hazards during crane operations, thereby assisting in improving safety management on site.

🌐 Live Demo: Crane Safety Risk Identification

Key Features
Automated Risk Detection: Utilizes computer vision models to identify and alert potential safety risks in tower crane operations.
Real-time Processing: Supports image/video input with real-time output of detection results and risk warnings.
Visualization: Visualizes risk areas and bounding boxes to facilitate quick decision-making for on-site personnel.
Web-based Interface: A lightweight, interactive web demo accessible directly from the browser.
